# 🌍 COVID-19 Global Data Analysis using Python

A data analysis project that collects, cleans, analyzes, and visualizes global COVID-19 statistics using Python. The project demonstrates web scraping, data preprocessing, exploratory data analysis (EDA), and data visualization techniques.

---

## 📌 Project Overview

This project scrapes COVID-19 statistics from the Worldometer website, processes the data using Pandas, and creates insightful visualizations to understand global COVID-19 trends.

The project focuses on:

- Web Scraping
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Feature Engineering

---

## 🚀 Features

- 🌐 Web scraping using **Requests** and **BeautifulSoup**
- 📄 Export scraped data to CSV
- 🧹 Clean and preprocess real-world data
- 📊 Analyze COVID-19 statistics by country
- 📈 Create interactive and static visualizations
- 📉 Compare total cases, recoveries, deaths, and active cases
- 📌 Generate summary insights

## 📥 Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/COVID-19-Data-Analysis.git
```

Go to the project directory:

```bash
cd COVID-19-Data-Analysis
```

Install the required packages:

```bash
pip install -r requirements.txt
```

Run the Jupyter Notebook:

```bash
jupyter notebook
```

## 📈 Key Insights

- The United States reported the highest number of confirmed COVID-19 cases.
- Recovery rates were high in many countries with large case counts.
- COVID-19 cases were concentrated in a relatively small number of countries.
- Visual analysis helps compare the impact across different regions.
